# 飞书文档格式化工具

这个工具帮助你处理文档中的空行，使其适合复制到飞书文档。

## 何时使用此工具

当用户出现以下情况时使用此工具：

- 提到需要复制文档到飞书
- 提到文档有空行，复制到飞书不好看
- 要求移除文档中的空行
- 要求格式化文档以便复制到飞书

## 核心功能

### 1. 移除空行

**处理规则**：
- 移除文档中的所有空行
- 保持文本内容的完整性
- 保留代码块和表格的格式

**处理步骤**：
1. 读取原始文档内容
2. 移除所有空行
3. 保持代码块和表格的结构
4. 输出处理后的内容

### 2. 示例

**原始文档**：
```markdown
# 标题

## 子标题

内容1

内容2

```

**处理后**：
```markdown
# 标题
## 子标题
内容1
内容2
```

### 3. 注意事项

- 此工具只会移除空行，不会修改其他内容
- 代码块和表格的格式会被保留
- 处理后的内容适合直接复制到飞书文档

## 使用示例

### 示例1：处理单个文档

```
用户: 帮我处理一下这个文档，移除空行以便复制到飞书
助手: 好的，我来帮你处理文档。请提供文档内容或文件路径。
用户: [文档内容]
助手: 已处理完成，以下是移除空行后的内容：
[处理后的内容]
```

### 示例2：处理指定文件

```

助手: 好的，请提供文件路径。

助手: 已处理完成，以下是移除空行后的内容：
[处理后的内容]
```

## 常见问题

**Q: 处理后的内容会影响代码格式吗？**
A: 不会，代码块和表格的格式会被保留。

**Q: 可以处理哪些格式的文档？**
A: 主要处理 Markdown 格式的文档，也可以处理纯文本。

**Q: 处理后的内容是否适合直接复制到飞书？**
A: 是的，处理后的内容没有空行，复制到飞书后会保持整洁的格式。
