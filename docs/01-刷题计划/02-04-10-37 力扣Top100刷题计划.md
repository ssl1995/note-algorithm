## 四、刷题策略

```mermaid
flowchart TD
    A[开始刷题] --> B{是否了解该类型算法}
    B -->|是| C[直接做题]
    B -->|否| D[先学习算法模板]
    D --> C
    C --> E{能否在30分钟内AC}
    E -->|是| F[记录解题思路]
    E -->|否| G[看题解学习]
    G --> H[理解后重做]
    H --> F
    F --> I[加入复习清单]
    I --> J{是否刷完本类型}
    J -->|是| K[进入下一类型]
    J -->|否| L[继续当前类型]
    L --> C
    K --> B
```

## 九、注意事项

1. **先理解再刷题**：每种题型先掌握基本模板，再做变形题
2. **限时练习**：简单题15分钟、中等题30分钟、困难题45分钟
3. **记录错题**：建立错题本，定期复习
4. **手写代码**：面试时需要手写，平时也要多练习
5. **总结规律**：同类型题目做完后总结通用解法
6. **Java特性**：熟练使用Java集合框架(HashMap/ArrayList/PriorityQueue等)
