# 长期训练规划

- 课程平台：B站

## 阶段1：进阶强化期（第1-4周）

### 目标
- 增加肌肉量
- 提升力量水平
- 改善身体形态

### 训练安排
- 每周训练6天，休息1天
- 训练分化：推→拉→腿（上下肢分化）
- 每组间休息90-120秒
- 采用超级组、递减组等进阶技术

### 每周重点
- 第1周：适应推拉腿分化，掌握超级组技术
- 第2周：增加训练重量，尝试更大负荷
- 第3周：引入递减组训练，提高训练强度
- 第4周：测试1RM，评估进步

---

## 阶段2：高阶训练期（第5-8周）

### 目标
- 进一步增肌
- 刻画肌肉线条
- 准备5月30日表姐婚礼

### 训练安排
- 每周训练6天，休息1天
- 训练分化：部位细分（胸上/胸下、背阔/中上背等）
- 每组间休息60-90秒（增肌）或30-60秒（雕刻线条）
- 采用巨人组、预先疲劳法等技术

### 每周重点
- 第5周：细分部位训练，提高训练针对性
- 第6周：引入预先疲劳法，增强肌肉刺激
- 第7周：提高训练频率，增加雕刻线条训练
- 第8周：调整状态，准备展示（5月30日）

## 睡眠改善计划

### 目标
- 逐步将入睡时间提前到23:00前
- 保证7-8小时睡眠
- 提高睡眠质量

### 具体措施
- 训练后避免使用电子设备（21:00后）
- 22:30开始准备入睡
- 睡前放松：阅读、冥想、拉伸
- 保证卧室黑暗、安静、凉爽
