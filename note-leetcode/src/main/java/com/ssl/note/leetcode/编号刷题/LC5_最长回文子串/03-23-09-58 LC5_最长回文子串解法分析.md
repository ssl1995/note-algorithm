# LC5_最长回文子串
## 题目描述
给你一个字符串 s，找到 s 中最长的回文子串。
示例1：
```
输入：s = "babad"
输出："bab"
解释："aba" 同样是符合题意的答案。
```
示例2：
```
输入：s = "cbbd"
输出："bb"
解释：回文串 "bb"。
```
## 解法概览
| 解法 | 思路 | 核心公式 | 时间复杂度 | 空间复杂度 |
|------|------|----------|------------|------------|
| 暴力枚举 | 枚举所有子串，判断是否回文 | `isPalindrome(i,j)` | O(n³) | O(1) |
| 中心扩展 | 以每个字符为中心向两边扩展 | `expand(i,i)` 和 `expand(i,i+1)` | O(n²) | O(1) |
| 动态规划 | DP表记录子串是否为回文 | `dp[i][j]=dp[i+1][j-1]&&s[i]==s[j]` | O(n²) | O(n²) |
| Manacher | 马拉车算法 | 预处理 + 贪心 | O(n) | O(n) |
## 记忆口诀
最长回文子串，中心扩展最简单；奇数偶数两种扩，扩展到不等为止
## 解法一：暴力枚举
### 思路
枚举所有子串，对每个子串判断是否为回文串，记录最长的回文子串。
### 核心公式
- 子串范围：`[i, j]`，长度 `j-i+1`
- 回文判断：`cs[i]==cs[j]` 逐字符对比
### 代码示例
```java
public String longestPalindrome(String s) {
    if (s.length() < 2) {
        return s;
    }
    int len = s.length();
    int maxLen = 1;
    int begin = 0;
    char[] cs = s.toCharArray();
    for (int i = 0; i < len - 1; i++) {
        for (int j = i + 1; j < len; j++) {
            if (j - i + 1 > maxLen && isPalindrome(cs, i, j)) {
                maxLen = j - i + 1;
                begin = i;
            }
        }
    }
    return s.substring(begin, begin + maxLen);
}

private boolean isPalindrome(char[] cs, int i, int j) {
    while (i < j) {
        if (cs[i] != cs[j]) {
            return false;
        }
        i++;
        j--;
    }
    return true;
}
```
### 复杂度分析
- 时间复杂度：O(n³)，枚举 O(n²) × 判断 O(n)
- 空间复杂度：O(1)
### 优缺点
- 优点：思路直接，易于理解
- 缺点：时间复杂度太高，会超时
## 解法二：中心扩展法（推荐）
### 思路
回文串的中心有两种：单字符中心（奇数长度如"aba"）或双字符中心（偶数长度如"abba"）。从每个可能的中心向两边扩展，记录最长的回文。
### 核心公式
- 奇数中心：`expand(i, i)` 以字符 i 为中心
- 偶数中心：`expand(i, i+1)` 以 i 和 i+1 为中心
- 扩展公式：相等时 `left--, right++`，停止时回文长度 `= right - left - 1`
### 图解过程
```
s = "babad"

以每个位置为中心扩展：

i=0 "b": 奇数 expand(0,0) → "b" 长度1
          偶数 expand(0,1) → ""  长度0
i=1 "a": 奇数 expand(1,1) → "bab" 长度3  ← 最长
          偶数 expand(1,2) → ""   长度0
i=2 "b": 奇数 expand(2,2) → "b"  长度1
          偶数 expand(2,3) → "aba" 长度3  ← 最长
i=3 "a": 奇数 expand(3,3) → "a"  长度1
          偶数 expand(3,4) → ""   长度0

答案："bab" 或 "aba"
```
### 代码示例
```java
public String longestPalindrome(String s) {
    if (s.length() < 2) {
        return s;
    }
    int len = s.length();
    int maxLen = 1;
    int begin = 0;
    char[] cs = s.toCharArray();

    for (int i = 0; i < len - 1; i++) {
        // 奇数长度：以 i 为中心
        int len1 = expand(cs, len, i, i);
        // 偶数长度：以 i 和 i+1 为中心
        int len2 = expand(cs, len, i, i + 1);
        int curLen = Math.max(len1, len2);

        if (curLen > maxLen) {
            maxLen = curLen;
            // 计算起始位置
            begin = i - (maxLen - 1) / 2;
        }
    }
    return s.substring(begin, begin + maxLen);
}

private int expand(char[] cs, int len, int left, int right) {
    while (left >= 0 && right < len && cs[left] == cs[right]) {
        left--;
        right++;
    }
    return right - left - 1;
}
```
### 复杂度分析
- 时间复杂度：O(n²)，每个位置最多扩展 O(n)
- 空间复杂度：O(1)
### 优缺点
- 优点：时间复杂度较低，代码简洁，面试推荐
- 缺点：最坏情况仍需 O(n²)
### 测试用例
- 示例1：s="babad" → "bab" 或 "aba"
- 示例2：s="cbbd" → "bb"
- 边界：s="a" → "a"
- 边界：s="ac" → "a" 或 "c"
## 解法三：动态规划
### 思路
用 dp[i][j] 表示 s[i..j] 是否为回文串。状态转移：如果是回文串，首尾字符相等且去掉首尾后仍是回文。
### 核心公式
- 初始化：`dp[i][i] = true`（单字符）
- 状态转移：`dp[i][j] = dp[i+1][j-1] && s[i] == s[j]`
- 长度2：`s[i] == s[i+1]`
### 代码示例
```java
public String longestPalindrome(String s) {
    if (s.length() < 2) {
        return s;
    }
    int len = s.length();
    int maxLen = 1;
    int begin = 0;
    boolean[][] dp = new boolean[len][len];

    for (int i = 0; i < len; i++) {
        dp[i][i] = true;
    }

    for (int j = 1; j < len; j++) {
        for (int i = 0; i < j; i++) {
            if (s.charAt(i) != s.charAt(j)) {
                dp[i][j] = false;
            } else {
                if (j - i < 3) {
                    dp[i][j] = true;
                } else {
                    dp[i][j] = dp[i + 1][j - 1];
                }
            }

            if (dp[i][j] && j - i + 1 > maxLen) {
                maxLen = j - i + 1;
                begin = i;
            }
        }
    }
    return s.substring(begin, begin + maxLen);
}
```
### 复杂度分析
- 时间复杂度：O(n²)
- 空间复杂度：O(n²)
### 优缺点
- 优点：思路清晰，体现 DP 思想
- 缺点：空间复杂度高，可用一维优化但复杂
## 解法四：Manacher 算法
### 思路
通过预处理将奇偶长度统一处理，再用贪心策略快速找到最长回文。面试了解即可。
### 复杂度分析
- 时间复杂度：O(n)
- 空间复杂度：O(n)
## 面试回答模板
