# LC1143_最长公共子序列
## 题目描述
给定两个字符串 text1 和 text2，返回这两个字符串的最长公共子序列的长度。如果不存在公共子序列，返回 0 。
子序列：不删除字符，保持相对顺序
示例1：
```
输入：text1 = "abcde", text2 = "ade"
输出：3
解释：最长公共子序列是 "ade"，长度为 3。
```
示例2：
```
输入：text1 = "abc", text2 = "abc"
输出：3
解释：最长公共子序列是 "abc"，长度为 3。
```
示例3：
```
输入：text1 = "abc", text2 = "def"
输出：0
解释：不存在公共子序列，返回 0。
```
## 解法概览
| 解法 | 思路 | 核心公式 | 时间复杂度 | 空间复杂度 |
|------|------|----------|------------|------------|
| 二维DP | 经典动态规划 | `dp[i][j]=max(dp[i-1][j],dp[i][j-1],dp[i-1][j-1]+1)` | O(m*n) | O(m*n) |
| 空间优化 | 滚动数组 | 只保留上一行和当前行 | O(m*n) | O(n) |
## 记忆口诀
公共子序列，DP来解决；字符相等加1，不等取较大
## 解法一：二维动态规划
### 思路
定义 dp[i][j] 表示 text1[0..i] 和 text2[0..j] 的最长公共子序列长度。根据最后一个字符是否相等来推导状态转移。
### 核心公式
- 初始化：`dp[i][0] = 0`，`dp[0][j] = 0`
- 状态转移：
  - `text1[i] == text2[j]`：`dp[i][j] = dp[i-1][j-1] + 1`
  - `text1[i] != text2[j]`：`dp[i][j] = max(dp[i-1][j], dp[i][j-1])`
### 图解过程
```
text1 = "abcde", text2 = "ade"

dp表：
        a   d   e
    [0] [0] [0] [0]
a   [0]  1   1   1
b   [0]  1   1   1
c   [0]  1   1   1
d   [0]  1   2   2
e   [0]  1   2   3

填充过程：
- i=3(j='d'): text1[3]='d'==text2[1]='d' → dp[3][1]=dp[2][0]+1=1
- i=4(j='e'): text1[4]='e'==text2[2]='e' → dp[4][2]=dp[3][1]+1=2
- 最终：dp[4][2]=3，即 "ade"
```
### 代码示例
```java
public int longestCommonSubsequence(String text1, String text2) {
    if (text1 == null || text2 == null || text1.length() == 0 || text2.length() == 0) {
        return 0;
    }
    char[] str1 = text1.toCharArray();
    char[] str2 = text2.toCharArray();
    int N = str1.length;
    int M = str2.length;
    int[][] dp = new int[N][M];

    // 初始化：i=0 或 j=0 的情况
    dp[0][0] = str1[0] == str2[0] ? 1 : 0;
    for (int j = 1; j < M; j++) {
        dp[0][j] = str1[0] == str2[j] ? 1 : dp[0][j - 1];
    }
    for (int i = 1; i < N; i++) {
        dp[i][0] = str1[i] == str2[0] ? 1 : dp[i - 1][0];
    }

    // 填表
    for (int i = 1; i < N; i++) {
        for (int j = 1; j < M; j++) {
            int p1 = dp[i - 1][j];              // 上面
            int p2 = dp[i][j - 1];              // 左边
            int p3 = str1[i] == str2[j] ? (1 + dp[i - 1][j - 1]) : 0;  // 左上+1
            dp[i][j] = Math.max(p1, Math.max(p2, p3));
        }
    }
    return dp[N - 1][M - 1];
}
```
### 复杂度分析
- 时间复杂度：O(m*n)，m 和 n 分别为两个字符串长度
- 空间复杂度：O(m*n)
### 优缺点
- 优点：思路清晰，状态定义明确
- 缺点：空间可以优化
### 测试用例
- 示例1：text1="abcde", text2="ade" → 3
- 示例2：text1="abc", text2="abc" → 3
- 示例3：text1="abc", text2="def" → 0
### 边界情况分析
- 空字符串：返回 0
- 长度为 1：直接比较
### 优化版本
```java
// 一维数组空间优化
public int longestCommonSubsequence(String text1, String text2) {
    if (text1.length() < text2.length()) {
        return longestCommonSubsequence(text2, text1);
    }
    int m = text1.length();
    int n = text2.length();
    int[] dp = new int[n + 1];

    for (int i = 1; i <= m; i++) {
        int prev = 0;
        for (int j = 1; j <= n; j++) {
            int temp = dp[j];
            if (text1.charAt(i - 1) == text2.charAt(j - 1)) {
                dp[j] = prev + 1;
            } else {
                dp[j] = Math.max(dp[j], dp[j - 1]);
            }
            prev = temp;
        }
    }
    return dp[n];
}
```
### 解题技巧总结
- 技巧1：子序列不要求连续，所以不等时取 max(上，左)
- 技巧2：dp 表多一行一列方便边界处理
- 技巧3：空间优化时需要保存左上角的值

## 面试回答模板
面试官好，这道题是经典的动态规划问题。核心思路是定义 dp[i][j] 表示两个字符串前缀的最长公共子序列长度。状态转移分两种情况：1）当两个字符串当前字符相等时，等于左上角加1；2）不等时，取上方或左方的较大值。这样从左上角填到右下角，最终 dp[m][n] 就是答案。时间复杂度 O(mn)，空间可以用滚动数组优化到 O(min(m,n))。
