# 快速排序详解与LC215应用
## 快速排序基本概念
快速排序（Quick Sort）是一种高效的排序算法，采用分治策略。其核心思想是通过一个基准元素（pivot）将数组分为两部分，使得左边的元素都小于pivot，右边的元素都大于等于pivot，然后对两部分分别递归进行快速排序。
## 快速排序步骤
1. 选择一个基准元素（pivot）
2. 通过partition函数将数组分为两部分
3. 递归对左右两部分进行排序
## partition函数详解
partition函数是快速排序的核心，它的目标是将数组分为两部分并返回pivot的最终位置。
### 常见的partition方法
#### 方法一：左右指针法
```java
private int partition(int[] nums, int left, int right) {
    // 选择最右边元素作为pivot
    int pivot = nums[right];
    int i = left - 1;
    
    for (int j = left; j < right; j++) {
        if (nums[j] < pivot) {
            i++;
            swap(nums, i, j);
        }
    }
    swap(nums, i + 1, right);
    return i + 1;
}
```
#### 方法二：前后指针法（本题使用）
```java
private int partition(int[] nums, int left, int right) {
    // 随机选择pivot，防止极端情况
    int random = new Random().nextInt(right - left + 1) + left;
    swap(nums, random, right);
    
    int p1 = left - 1;
    int p2 = left;
    
    while (p2 <= right) {
        if (nums[p2] < nums[right]) {
            p1++;
            swap(nums, p1, p2);
        }
        p2++;
    }
    
    p1++;
    swap(nums, p1, right);
    return p1;
}
```
## 图解partition过程
以数组 [3,6,4,2,1,5] 为例，pivot选择最右边的5：
1. 初始：p1=-1, p2=0, nums[right]=5
2. p2=0: nums[0]=3 < 5, p1=0, swap(0,0), 数组不变
3. p2=1: nums[1]=6 >= 5, 不操作
4. p2=2: nums[2]=4 < 5, p1=1, swap(1,2), 数组变为 [3,4,6,2,1,5]
5. p2=3: nums[3]=2 < 5, p1=2, swap(2,3), 数组变为 [3,4,2,6,1,5]
6. p2=4: nums[4]=1 < 5, p1=3, swap(3,4), 数组变为 [3,4,2,1,6,5]
7. p2=5: 循环结束，p1+1=4, swap(4,5), 数组变为 [3,4,2,1,5,6]
8. 返回p1+1=4，pivot位置为4
## 快速选择算法（解决LC215）
快速选择（Quick Select）是快速排序的变形，用于在无序数组中查找第K小（或第K大）元素的算法。
### 与快速排序的区别
- 快速排序：对两部分都进行递归排序
- 快速选择：只对目标所在的部分进行递归
### 核心思路
1. 使用partition函数将数组分为两部分
2. 根据pivot位置与目标位置的关系，决定搜索方向
3. 重复直到找到目标位置
### 代码实现
```java
public int findKthLargest(int[] nums, int k) {
    // 第k大元素等价于第n-k小元素
    int target = nums.length - k;
    int left = 0;
    int right = nums.length - 1;
    
    int pivot = partition(nums, left, right);
    while (pivot != target) {
        if (pivot < target) {
            // 目标在右半部分
            left = pivot + 1;
        } else {
            // 目标在左半部分
            right = pivot - 1;
        }
        pivot = partition(nums, left, right);
    }
    
    return nums[pivot];
}
```
## 时间复杂度分析
### 快速排序
- 平均时间复杂度：O(nlogn)
- 最坏时间复杂度：O(n^2)（当数组有序或逆序时）
- 空间复杂度：O(logn)（递归栈）
### 快速选择
- 平均时间复杂度：O(n)
- 最坏时间复杂度：O(n^2)
- 空间复杂度：O(1)
## 优化方法
### 1. 随机化pivot选择
```java
int random = new Random().nextInt(right - left + 1) + left;
swap(nums, random, right);
```
作用：避免数组有序或逆序时的最坏情况
### 2. 三数取中法
```java
int mid = left + (right - left) / 2;
if (nums[left] > nums[right]) swap(nums, left, right);
if (nums[mid] > nums[right]) swap(nums, mid, right);
if (nums[left] > nums[mid]) swap(nums, left, mid);
swap(nums, mid, right);
```
作用：选择大小适中的pivot，提高效率
### 3. 小数组使用插入排序
当数组规模较小时，插入排序效率更高
## 面试常见问题
1. **快速排序是稳定排序吗？**
   答：不是稳定排序，因为相等的元素可能被交换位置。
2. **快速排序的平均时间复杂度为什么是O(nlogn)？**
   答：每次partition将数组分为两部分，总共有logn层，每层需要O(n)时间。
3. **快速排序最坏情况是什么？**
   答：数组有序或逆序时，每次pivot都是最大或最小值，时间复杂度退化为O(n^2)。
4. **如何优化快速排序？**
   答：随机化pivot选择、三数取中法、小数组用插入排序。
## 总结
快速排序是一种高效的排序算法，其核心在于partition函数。通过理解partition函数的分割机制，我们可以将其应用到快速选择算法中，解决Top-K类问题。快速选择算法只需要对目标所在的部分进行递归，平均时间复杂度为O(n)，是解决LC215这类问题的最优解法。
