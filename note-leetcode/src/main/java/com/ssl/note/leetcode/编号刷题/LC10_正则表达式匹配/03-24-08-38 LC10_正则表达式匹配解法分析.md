# LC10_正则表达式匹配
## 题目描述
给你一个字符串 s 和一个字符规律 p，请你来实现一个支持 '.' 和 '*' 的正则表达式匹配。
- '.' 匹配任意单个字符
- '*' 匹配零个或多个前面的那一个元素
示例1：
```
输入：s = "aa", p = "a"
输出：false
解释："a" 无法匹配 "aa" 整个字符串。
```
示例2：
```
输入：s = "aa", p = "a*"
输出：true
解释：'*' 可以匹配零个或多个前面的那一个元素，这里前面的元素是 'a'。因此，字符串 "aa" 可被视为 'a' 重复了一次。
```
示例3：
```
输入：s = "ab", p = ".*"
输出：true
解释：".*" 表示可匹配零个或多个（'*'）任意字符（'.'）。
```
## 解法概览
| 解法 | 思路 | 核心公式 | 时间复杂度 | 空间复杂度 |
|------|------|----------|------------|------------|
| 动态规划 | 状态转移 | `dp[i][j]` 表示 s 前 i 个字符和 p 前 j 个字符是否匹配 | O(m*n) | O(m*n) |
| 递归 + 记忆化 | 回溯 | 分情况处理 '*' | O(m*n) | O(m*n) |
## 记忆口诀
正则匹配，动态规划；星号三种情况，点号匹配任意
## 解法一：动态规划
### 思路
定义 dp[i][j] 表示 s 的前 i 个字符和 p 的前 j 个字符是否匹配。根据 p 的当前字符是否为 '*' 来分情况处理。
### 核心公式
- 初始化：`dp[0][0] = true`（空串匹配）
- 首行初始化：`dp[0][j] = dp[0][j-2] && p[j-1] == '*'`（处理 `a*` 形式的模式）
- 状态转移：
  - `p[j-1] == '*'`：
    - 情况1：* 匹配 0 次 → `dp[i][j] = dp[i][j-2]`
    - 情况2：* 匹配 1+ 次 → `dp[i][j] = dp[i-1][j] && (s[i-1] == p[j-2] || p[j-2] == '.')`
  - `p[j-1] != '*'`：
    - 情况1：字符相等 → `dp[i][j] = dp[i-1][j-1] && s[i-1] == p[j-1]`
    - 情况2：遇到 '.' → `dp[i][j] = dp[i-1][j-1] && p[j-1] == '.'`
### 图解过程
```
s = "aaa", p = "ab*.*"

dp表：
        a   b   *   .   *
    [T] [F] [F] [F] [F] [F]
a   [F] [T] [F] [T] [F] [T]
a   [F] [F] [F] [F] [T] [T]
a   [F] [F] [F] [F] [F] [T]

填充过程：
- dp[0][2]："b*" 匹配空串 → true
- dp[1][2]："a" 匹配 "ab*" → true（* 匹配 0 次）
- dp[1][4]："a" 匹配 "ab*.*" → true（.* 匹配 0 次）
- 最终：dp[3][5] = true
```
### 代码示例
```java
public boolean isMatch(String s, String p) {
    int m = s.length();
    int n = p.length();
    // dp[i][j]表示s前i个字符和p前j个字符是否匹配
    boolean[][] dp = new boolean[m + 1][n + 1];

    // 初始化：空串匹配空串
    dp[0][0] = true;

    // 初始化首行：处理a*b*c*形式的模式
    for (int j = 2; j <= n; j += 2) {
        dp[0][j] = dp[0][j - 2] && p.charAt(j - 1) == '*';
    }

    // 填表
    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (p.charAt(j - 1) == '*') {
                // 情况1：* 匹配0次
                if (dp[i][j - 2]) {
                    dp[i][j] = true;
                }
                // 情况2：* 匹配1+次，且当前字符匹配
                else if (dp[i - 1][j] && (s.charAt(i - 1) == p.charAt(j - 2) || p.charAt(j - 2) == '.')) {
                    dp[i][j] = true;
                }
            } else {
                // 情况1：字符相等
                if (dp[i - 1][j - 1] && s.charAt(i - 1) == p.charAt(j - 1)) {
                    dp[i][j] = true;
                }
                // 情况2：遇到'.'（匹配任意字符）
                else if (dp[i - 1][j - 1] && p.charAt(j - 1) == '.') {
                    dp[i][j] = true;
                }
            }
        }
    }

    return dp[m][n];
}
```
### 复杂度分析
- 时间复杂度：O(m*n)，m 和 n 分别为 s 和 p 的长度
- 空间复杂度：O(m*n)
### 优缺点
- 优点：思路清晰，状态转移明确，适合面试
- 缺点：空间复杂度较高，可优化为 O(n)
### 测试用例
- 示例1：s="aa", p="a" → false
- 示例2：s="aa", p="a*" → true
- 示例3：s="ab", p=".*" → true
- 示例4：s="aaa", p="ab*.*" → true
### 边界情况分析
- 空字符串：s="", p="" → true
- 空模式：s="a", p="" → false
- 星号开头：p="*a" → 无效，星号必须跟在字符后面
- 连续星号：p="a**" → 无效，星号后面必须有字符
### 优化版本
```java
// 空间优化到 O(n)
public boolean isMatch(String s, String p) {
    int m = s.length();
    int n = p.length();
    boolean[] dp = new boolean[n + 1];

    dp[0] = true;
    for (int j = 2; j <= n; j += 2) {
        dp[j] = dp[j - 2] && p.charAt(j - 1) == '*';
    }

    for (int i = 1; i <= m; i++) {
        boolean prev = dp[0];
        dp[0] = false;
        for (int j = 1; j <= n; j++) {
            boolean temp = dp[j];
            if (p.charAt(j - 1) == '*') {
                dp[j] = dp[j - 2] || (dp[j] && (s.charAt(i - 1) == p.charAt(j - 2) || p.charAt(j - 2) == '.'));
            } else {
                dp[j] = prev && (s.charAt(i - 1) == p.charAt(j - 1) || p.charAt(j - 1) == '.');
            }
            prev = temp;
        }
    }
    return dp[n];
}
```
### 解题技巧总结
- 技巧1：多一行一列处理边界情况（空字符串）
- 技巧2：遇到 '*' 时，分匹配 0 次和匹配 1+ 次两种情况
- 技巧3：'.' 匹配任意字符，视为字符相等
