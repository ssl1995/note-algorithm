# 位运算详解：异或与常用技巧
## 一、位运算基础概念
位运算是直接对二进制位进行操作的运算，效率极高，在算法中经常用于优化时间和空间复杂度。
| 运算符 | 名称 | 功能 | 示例 |
|--------|------|------|------|
| `&` | 与 | 两个位都为1才为1 | `1010 & 1100 = 1000` |
| `|` | 或 | 两个位有一个为1就为1 | `1010 | 1100 = 1110` |
| `^` | 异或 | 两个位不同才为1 | `1010 ^ 1100 = 0110` |
| `~` | 非 | 取反 | `~1010 = 0101`（补码表示） |
| `<<` | 左移 | 左移n位，右边补0 | `1010 << 2 = 101000` |
| `>>` | 右移 | 右移n位，左边补符号位 | `1010 >> 2 = 0010` |
| `>>>` | 无符号右移 | 右移n位，左边补0 | `1010 >>> 2 = 0010` |
## 二、异或运算详解
### 1. 异或的定义
异或（XOR）运算的规则是：相同为0，不同为1。
### 2. 异或的性质
- **交换律**：`a ^ b = b ^ a`
- **结合律**：`(a ^ b) ^ c = a ^ (b ^ c)`
- **自反性**：`a ^ a = 0`
- **恒等性**：`a ^ 0 = a`
- **逆运算**：`a ^ b ^ b = a`
### 3. 异或的应用场景
- **找唯一**：如 LC136 只出现一次的数字
- **交换变量**：无需临时变量 `a = a ^ b; b = a ^ b; a = a ^ b;`
- **检测两个数是否同号**：`(a ^ b) >= 0` 同号，否则异号
- **翻转位**：与 1 异或翻转，与 0 异或保持不变
## 三、常见位运算技巧
### 1. 基础技巧
| 技巧 | 代码 | 说明 |
|------|------|------|
| 检查奇偶 | `n & 1 == 1` | 最低位为1是奇数 |
| 取最低位1 | `n & -n` | 如 6(110) → 2(10) |
| 清除最低位1 | `n & (n-1)` | 如 6(110) → 4(100) |
| 检测2的幂 | `(n & (n-1)) == 0` | 二进制只有一个1 |
| 计算二进制中1的个数 | 循环 `n &= n-1` | 每次清除最低位1 |
| 取模2的幂 | `n & (2^k - 1)` | 如 n & 15 相当于 n % 16 |
| 符号位提取 | `n >> 31` | 正数为0，负数为-1 |
### 2. 中级技巧
| 技巧 | 代码 | 说明 |
|------|------|------|
| 两数交换 | `a ^= b; b ^= a; a ^= b;` | 无需临时变量 |
| 绝对值 | `(n ^ (n >> 31)) - (n >> 31)` | 负数取反加1 |
| 最大值 | `b & ((a-b) >> 31) | a & (~(a-b) >> 31)` | 不使用if-else |
| 最小值 | `a & ((a-b) >> 31) | b & (~(a-b) >> 31)` | 不使用if-else |
### 3. 高级技巧
| 技巧 | 说明 | 应用 |
|------|------|------|
| 位掩码 | 用特定二进制模式提取或设置位 | 权限管理、标志位 |
| 位压缩 | 将多个布尔值存储在一个整数中 | 节省空间 |
| 位操作优化 | 替代乘除、取模等运算 | 提高性能 |
## 四、算法题中的应用
### 1. 只出现一次的数字（LC136）
```java
public int singleNumber(int[] nums) {
    int single = 0;
    for (int num : nums) {
        single ^= num;  // 异或运算，两两抵消
    }
    return single;
}
```
### 2. 只出现一次的数字 II（LC137）
```java
public int singleNumber(int[] nums) {
    int ones = 0, twos = 0;
    for (int num : nums) {
        ones = (ones ^ num) & ~twos;
        twos = (twos ^ num) & ~ones;
    }
    return ones;
}
```
### 3. 只出现一次的数字 III（LC260）
```java
public int[] singleNumber(int[] nums) {
    int xor = 0;
    for (int num : nums) {
        xor ^= num;
    }
    int mask = xor & -xor;  // 取最低位1
    int a = 0, b = 0;
    for (int num : nums) {
        if ((num & mask) != 0) {
            a ^= num;
        } else {
            b ^= num;
        }
    }
    return new int[]{a, b};
}
```
### 4. 二进制中1的个数（LC191）
```java
public int hammingWeight(int n) {
    int count = 0;
    while (n != 0) {
        n &= n - 1;  // 清除最低位1
        count++;
    }
    return count;
}
```
### 5. 缺失的数字（LC268）
```java
public int missingNumber(int[] nums) {
    int missing = nums.length;
    for (int i = 0; i < nums.length; i++) {
        missing ^= i ^ nums[i];
    }
    return missing;
}
```
## 五、位运算面试常见问题
### 1. 异或的性质及应用
**问题**：异或有哪些性质？如何用异或找到数组中只出现一次的元素？
**回答**：异或的性质包括交换律、结合律、自反性（a^a=0）和恒等性（a^0=a）。找唯一元素时，遍历数组异或所有元素，出现两次的元素会互相抵消，最终结果就是只出现一次的元素。
### 2. 位运算优化
**问题**：如何用位运算优化乘法和除法？
**回答**：
- 乘以2的幂：`n << k`
- 除以2的幂：`n >> k`
- 取模2的幂：`n & (2^k - 1)`
### 3. 位掩码的应用
**问题**：如何用位掩码表示权限？
**回答**：每个权限对应一个位，如：
- 读权限：0001 (1)
- 写权限：0010 (2)
- 执行权限：0100 (4)
- 管理权限：1000 (8)
组合权限用或运算，检查权限用与运算。
## 六、总结
位运算是算法优化的重要工具，掌握常见的位运算技巧可以：

2. **节省内存空间**：位压缩可以存储更多信息
3. **解决特定问题**：如找唯一元素、位计数等
4. **展示编程能力**：在面试中使用位运算可以体现对底层原理的理解
