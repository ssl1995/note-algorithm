# 力扣刷题助手

这个skill帮助你管理力扣刷题，根据题号或题目名自动查找或创建题目目录，并联网生成完整的题解文档。

## 何时使用此Skill

当用户出现以下情况时使用此skill：

- 提供力扣题号（如"LC32"、"32"、"第32题"）
- 提供力扣题目名称（如"最长有效括号"）
- 同时提供题号和题目名（如"LC62 不同路径"）
- 问"这道题怎么做"、"帮我分析这道题"
- 想开始刷一道新题

## 核心功能

### 1. 查找/创建题目目录

**题目目录命名规范**：`LC{编号}_{题目名称}`

**搜索路径**：`note-leetcode/src/main/java/com/ssl/note/leetcode/编号刷题/`

**查找/创建步骤**：
1. 用户输入题号或题目名（或两者都有）
2. 在编号刷题目录下搜索匹配的目录
3. 如果找到，进入下一步检查题解文档
4. 如果没找到，创建新目录并生成基础文件

### 2. 自动生成完整题解文档

**题解文档命名规范**：`{月-日-时-分} {题目名称}解法分析.md`

**题解检查步骤**：
1. 在题目目录下查找 `.md` 文件
2. 如果没有找到题解文档，执行以下步骤：
   - 联网搜索力扣题目信息（题目描述、难度、标签）
   - 搜索题解思路和算法分析
   - 生成完整的题解文档（无空行，适合飞书）
3. 如果已有题解文档，告知用户已有文档

**题解文档生成流程**：
```
用户输入 → 查找题目目录 → 检查题解
    ↓
    无题解 → 联网搜索题目信息 → 搜索题解思路 → 生成完整文档
```

### 3. 联网搜索策略

**搜索内容**：
1. **题目信息**：力扣官网题目描述、难度、标签
2. **题解思路**：常见解法、算法思路、解题技巧
3. **代码实现**：Java代码示例、边界情况处理

**搜索关键词**：
- "力扣 {题号} {题目名} 题解"
- "LeetCode {题号} solution Java"
- "{题目名} 动态规划/贪心/二分查找等算法"

**信息整合**：
- 从多个来源提取关键信息
- 整理成标准化的题解格式
- 确保内容准确、易懂

### 4. 完整工作流程

```
用户输入 → 查找题目目录
    ↓
    ├─ 找到目录 → 检查题解文档
    │                ├─ 有题解 → 告知用户
    │                └─ 无题解 → 联网搜索 → 生成完整题解
    │
    └─ 未找到 → 创建目录 → 创建Solution.java → 联网搜索 → 生成完整题解
```

## 注意事项

1. 题号格式统一使用 `LC{编号}`，如 LC32、LC300
2. 目录名称使用中文题目名，如 `LC32_最长有效括号`
3. 题解文档命名格式：`月-日-时-分 题目名称解法分析.md`
4. **文档无空行**：生成的题解文档移除所有空行，适合直接复制到飞书
5. 自动创建题解时，会使用当前北京时间生成文件名
6. 联网搜索可能需要几秒钟，请耐心等待
7. 生成的题解包含完整信息：题目描述、解法思路、代码、复杂度分析
